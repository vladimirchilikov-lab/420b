# 420 Beans — Пълни инструкции за деплой
## Complete Setup & Deployment Guide

---

## СТЪПКА 1: РЕГИСТРАЦИЯ И АКАУНТИ / STEP 1: ACCOUNTS

### A) Stripe (задължително / required)
1. Отиди на https://stripe.com → **Create account**
2. Попълни бизнес информацията
3. За тестване — **не** е нужна верификация

### B) SendGrid (за имейли / for emails)
1. Отиди на https://sendgrid.com → **Start for free**
2. Free план = 100 имейла/ден (достатъчно за старт)
3. Верифицирай **Sender Identity** (твоят FROM имейл)

### C) Netlify (хостинг / hosting)
1. Отиди на https://netlify.com → **Sign up**
2. Можеш да влезеш с GitHub акаунт

---

## СТЪПКА 2: STRIPE API КЛЮЧОВЕ / STEP 2: STRIPE API KEYS

### Тест ключове (за разработка / test keys):
1. Влез в https://dashboard.stripe.com
2. Горе вдясно → **Developers** → **API keys**
3. Копирай:
   - **Publishable key** → `pk_test_...` (не се ползва в кода — само ако добавяш Stripe.js)
   - **Secret key** → `sk_test_...` ← **САМО в Netlify env vars!**

### Live ключове (за реални плащания / production):
1. Активирай акаунта (попълни бизнес данни, банкова сметка)
2. Горе вляво → превключи от **Test mode** на **Live mode**
3. Вземи **Secret key** → `sk_live_...`

> ⚠️  **ВАЖНО**: Secret key НИКОГА не трябва да е в HTML/JS файлове!
> Поставя се САМО в Netlify Environment Variables!

---

## СТЪПКА 3: SENDGRID API КЛЮЧove / STEP 3: SENDGRID SETUP

1. Влез в https://app.sendgrid.com
2. Ляво меню → **Settings** → **API Keys** → **Create API Key**
3. Избери **Restricted Access** → включи **Mail Send** → **Full Access**
4. Копирай ключа → `SG.xxxxx` (показва се само веднъж!)

### Верифицирай изпращача (задължително!):
1. **Settings** → **Sender Authentication**
2. **Verify a Single Sender** → въведи имейла, от който ще изпращаш
3. Провери имейла и кликни линка за верификация

## СТЪПКА 4: ДЕПЛОЙ В NETLIFY / STEP 4: DEPLOY TO NETLIFY

### Метод A: Чрез GitHub (препоръчително)
1. Качи папката `420beans/` в GitHub repo
2. В Netlify → **Add new site** → **Import an existing project**
3. Свържи GitHub → избери репото
4. Build settings:
   - **Base directory**: (остава празно)
   - **Build command**: (остава празно)
   - **Publish directory**: `public`
5. **Deploy site**

### Метод B: Drag & Drop
1. В Netlify → **Sites** → **Deploy manually**
2. Плъзни цялата папка `420beans/` в браузъра
3. Готово!

---

## СТЪПКА 5: ENVIRONMENT VARIABLES В NETLIFY

1. В Netlify → твоят сайт → **Site configuration** → **Environment variables**
2. Добави следните:

| Key | Value | Описание |
|-----|-------|---------|
| `STRIPE_SECRET_KEY` | `sk_live_...` или `sk_test_...` | Stripe secret key |
| `STRIPE_WEBHOOK_SECRET` | `whsec_...` | Webhook signing secret (виж Стъпка 6) |
| `SENDGRID_API_KEY` | `SG.xxxxx` | SendGrid API key |
| `ADMIN_EMAIL` | `tvoiqt@email.com` | Твоят имейл за нотификации |
| `FROM_EMAIL` | `orders@420beans.com` | Верифициран изпращач в SendGrid |
| `SITE_URL` | `https://420beans.netlify.app` | URL-ът на сайта ти |

3. След добавяне → **Trigger deploy** (за да влязат в сила)

## СТЪПКА 6: STRIPE WEBHOOK / STEP 6: STRIPE WEBHOOK

### A) Намери webhook URL-а:
```
https://ТВОЯТ-NETLIFY-ДОМЕЙН.netlify.app/.netlify/functions/stripe-webhook
```
Или ако ползваш `/api/` пренасочването:
```
https://ТВОЯТ-ДОМЕЙН.netlify.app/api/stripe-webhook
```

### B) Добави webhook в Stripe:
1. https://dashboard.stripe.com → **Developers** → **Webhooks**
2. **Add endpoint** → въведи URL-а отгоре
3. **Select events** → избери: `checkout.session.completed`
4. **Add endpoint**
5. Кликни на новия webhook → **Signing secret** → **Reveal**
6. Копирай `whsec_...` → добави в Netlify env vars като `STRIPE_WEBHOOK_SECRET`

## СТЪПКА 7: ТЕСТВАНЕ / STEP 7: TESTING

### Тест карти на Stripe:
| Карта | Резултат |
|-------|---------|
| `4242 4242 4242 4242` | Успешно плащане |
| `4000 0000 0000 9995` | Отказано плащане |
| `4000 0025 0000 3155` | Изисква 3D Secure |

- Expiry: всяка бъдеща дата (напр. `12/34`)
- CVV: всякакви 3 цифри
- Zip: всякакви 5 цифри

### Тествай webhook локално (опционално):
```bash
# Инсталирай Stripe CLI
stripe listen --forward-to localhost:8888/.netlify/functions/stripe-webhook
```

## СТЪПКА 8: СОБСТВЕН ДОМЕЙН / STEP 8: CUSTOM DOMAIN

1. Netlify → сайта → **Domain management** → **Add custom domain**
2. Въведи домейна (напр. `420beans.com`)
3. Обнови DNS записите при регистратора:
   - CNAME: `www` → `твоят-netlify-id.netlify.app`
   - A record: `@` → Netlify IP (показва се)
4. Netlify автоматично добавя **SSL сертификат** (безплатно)
5. **Обнови `SITE_URL`** в env vars с новия домейн!
6. **Обнови webhook URL** в Stripe с новия домейн!

## ПРОМЯНА НА ЦЕНИ / CHANGING PRICES

Цените се дефинират на **две места** — трябва да ги промениш и на двете:

### 1. `netlify/functions/create-checkout-session.js` (редове ~17-29):
```javascript
const PRODUCTS = {
  p1: { price: 2800,  // 28.00 лв → промени тук
  p2: { price: 3200,  // 32.00 лв
  p3: { price: 2600,  // 26.00 лв
```
Цените са в **стотинки** (лв × 100)!

### 2. `public/index.html` — в JavaScript каталога (~долу в `<script>`):
```javascript
const catalog = {
  p1: { price: 28,   // промени тук също
  p2: { price: 32,
  p3: { price: 26,
```

## ДОБАВЯНЕ НА НОВ ПРОДУКТ / ADDING A PRODUCT

### В `create-checkout-session.js`:
```javascript
p4: {
  name_en: "Kenya Sunrise",
  name_bg: "Кения Изгрев",
  price: 3400,     // 34.00 лв
  currency: "bgn",
},
```

### В `index.html` каталога:
```javascript
p4: { en:"Kenya Sunrise", bg:"Кения Изгрев", price:34, color:"linear-gradient(170deg,#D4A96A,#8B5E3C)" },
```

После добави HTML карта по същия модел като p1/p2/p3.

---

## BGN vs EUR

Stripe поддържа BGN директно. Ако искаш EUR:
1. Смени `currency: "bgn"` на `currency: "eur"` в `create-checkout-session.js`
2. Смени `лв` на `€` в `index.html`
3. Цените остават в стотинки (центове)

## ЧЕСТА ГРЕШКА / COMMON ISSUES

| Проблем | Решение |
|---------|---------|
| "Payment initialization failed" | Провери `STRIPE_SECRET_KEY` в Netlify env vars |
| Имейлите не пристигат | Провери `SENDGRID_API_KEY` и **Sender Verification** в SendGrid |
| Webhook не работи | Провери `STRIPE_WEBHOOK_SECRET` — трябва да е от **конкретния** webhook endpoint |
| 404 на /api/ | Провери `netlify.toml` redirects и че functions са в `netlify/functions/` |
| BGN не се приема | Активирай BGN в Stripe Dashboard → Settings → Business settings → Bank accounts |

---

## ПОДДРЪЖКА НА STRIPE BGN

За да приемаш BGN:
1. Stripe Dashboard → **Settings** → **Business settings**
2. **Bank accounts and scheduling** → Add Bulgarian bank account (IBAN)
3. Stripe автоматично ще активира BGN settlements
